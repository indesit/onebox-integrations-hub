```cron
0 8 * * * export PYTHONPATH=/root/projects/onebox-integrations-hub && \

  >> /root/projects/onebox-integrations-hub/data/catalog_sync.log 2>&1
```

**Порядок виконання:**
```
08:00 → catalog_sync   (dim_product_variants актуальний)

```
